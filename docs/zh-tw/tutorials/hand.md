要獲取帶有 NBT 資料/組件的物品 ID，你可以使用 /kjs hand 指令（1.19 之前的版本使用 /kjs_hand）。
以下是使用方法：

1. 將物品放在主手中
2. 執行指令
3. 現在你應該會看到類似這樣的內容：
   ![](/docs/zh-tw/tutorials/hand/command_output.png)
4. 點擊綠色文字（物品 ID）來複製它
