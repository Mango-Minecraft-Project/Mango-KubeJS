## `KeyModifier_`

| 屬性名稱      | 說明                   |
| ------------- | ---------------------- |
| `"alt"`       | ALT 鍵                 |
| `"control"`   | CONTROL 鍵             |
| `"shift"`     | SHIFT 鍵               |
| `"none"`      | 無修飾按鍵             |
| [`KeyModifier`][KeyModifier] | 其他額外導入的修飾按鍵 |

[KeyModifier]: ./utils/KeyModifier
