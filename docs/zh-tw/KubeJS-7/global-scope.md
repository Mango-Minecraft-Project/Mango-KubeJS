## 常數與物件

| 名稱      | 類型                  | 描述                                                                             |
| --------- | --------------------- | -------------------------------------------------------------------------------- |
| `global`  | `Map<String, Object>` | 全域對象，您可以使用它在啟動腳本中設定對象和回調，並從伺服器或客戶端腳本引用它們 |
| `console` | `Console`             | 控制台對象，用於記錄日誌                                                         |
| `SECOND`  | `number`              | 1000（毫秒）                                                                     |
| `MINUTE`  | `number`              | 60000（毫秒，60 * SECOND）                                                       |
| `HOUR`    | `number`              | 3600000（毫秒，60 * MINUTE）                                                     |
