## `KeyModifier_`

| 属性名称      | 说明                   |
| ------------- | ---------------------- |
| `"alt"`       | ALT 键                 |
| `"control"`   | CONTROL 键             |
| `"shift"`     | SHIFT 键               |
| `"none"`      | 无修饰按键             |
| [`KeyModifier`][KeyModifier] | 其其他额外导入的修饰按键 |

[KeyModifier]: ./utils/KeyModifier
